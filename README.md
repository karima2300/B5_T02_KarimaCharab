# Projecte de Programació - Documentació amb Pydoc

## Introducció

Aquest projecte conté tres mòduls Python amb funcions utilitàries per a diferents tasques. L'objectiu principal és generar documentació automàtica utilitzant Pydoc.

**Fitxers inclosos:**
- `calculadora.py`: Funcions matemàtiques bàsiques (suma, resta, multiplicació, divisió, potència)
- `gestio_llibre.py`: Sistema de gestió d'una biblioteca de llibres
- `validacions.py`: Funcions per validar emails, telèfons, DNI, contrasenyes, etc.

## Instal·lació ràpida
No he fet res de instal·lacions

### Justificació

Aquest projecte no requereix cap procés d'instal·lació addicional perquè utilitza **únicament la llibreria estàndard de Python**. Els tres mòduls (`calculadora.py`, `gestio_llibre.py`, `validacions.py`) estan escrits amb funcions nadiues de Python i no depenen de cap paquet extern com Django, Flask o altres frameworks.

Per tant, l'única dependència necessària és tenir Python 3 instal·lat al sistema, que és el requisit mínim per executar qualsevol script Python.
